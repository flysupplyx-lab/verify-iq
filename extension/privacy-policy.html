Verify.IQ Privacy Policy

Last updated: February 11, 2026

TL;DR: Verify.IQ does not collect, store, or sell any personal data. All analysis happens in real-time and results are returned immediately. Nothing is logged or retained.

1. Information We Process

Verify.IQ processes the following data only when you explicitly submit it:

  • Email addresses — sent to our API for verification when you click "Verify"
  • Text content — sent to our API for AI detection when you click "Analyze"

2. How We Use Your Data

Data is processed in real-time to return verification/detection results. We do not:

  • Store or log submitted emails or text
  • Track your browsing activity
  • Collect analytics or usage data
  • Share data with third parties
  • Use data for advertising

3. Data Storage

The extension stores the following locally on your device via chrome.storage.local:

  • Your API server URL (default: localhost)
  • Your API key (if you have one)

This data never leaves your device and is only used to connect to the API.

4. Permissions

The extension requests the following Chrome permissions:

  • activeTab — to read selected text on the current page when you right-click
  • storage — to save your API URL and key locally
  • contextMenus — to add right-click menu options
  • scripting — to detect selected text when the popup opens

5. Third-Party Services

The extension connects to the Verify.IQ API (self-hosted or our servers) to process verification requests. No other third-party services are used.

6. Contact

For privacy questions, email: privacy@verifyiq.dev
